# Production Monitoring Stack

This project demonstrates a complete production-style monitoring and logging stack using:

- AWS EC2
- Nginx
- Prometheus
- Grafana
- Loki
- Promtail
- Alertmanager
- Node Exporter

## Project Goal

Build a centralized monitoring, logging, and alerting platform for a web application hosted on Nginx.
